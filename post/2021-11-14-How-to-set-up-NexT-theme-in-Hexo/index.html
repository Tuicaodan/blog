How to set up NexT theme in Hexo

Posted on 2021-11-14 In Hexo Views: Disqus:
Symbols count in article: 3.5k Reading time ≈ 3 mins.

The Hexo came with a default theme which is okay. You can design your own theme or use other users’ designs if you want to make it easy. Currently, there are more than 300+ themes posted here. I found some of the themes were stopped to update by the authors a couple of years ago. One of the parameters when I picked the theme, one of the parameters is that the last update should be within a year ago. So there might be less possibility of compatible issues with the newer versions of Hexo.

I decided to go with the NexT theme since it really aligns with my minimalism guideline. Here is the repo.
